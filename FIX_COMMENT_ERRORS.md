# Решение проблем после установки комментариев

## ❌ Ошибка 1: Uncaught SyntaxError: Unexpected identifier 'код'

**Причина:** Обратные кавычки `` ` `` в HTML ломали синтаксис.

**Решение:** ✅ Исправлено в index.html строка 2989

Было:
```html
Поддерживается: **жирный**, *курсив*, `код`, > цитата
```

Стало:
```html
Поддерживается: **жирный**, *курсив*, код, > цитата
```

---

## ❌ Ошибка 2: login is not defined

**Причина:** Кэш браузера показывает старую версию страницы.

**Решение:**

### Вариант 1: Полная перезагрузка
**Windows:** Ctrl + F5  
**Mac:** Cmd + Shift + R  
**Linux:** Ctrl + Shift + R

### Вариант 2: Очистка кэша
1. F12 → Console
2. Правый клик на кнопке обновления → "Очистить кэш и перезагрузить"

### Вариант 3: Режим инкогнито
Ctrl + Shift + N → открыть приложение заново

---

## ❌ Ошибка 3: OneSignalSDK blocked

**Причина:** Блокировщики рекламы или расширение Privacy Badger.

**Решение:**
- Отключите блокировщики для localhost
- Это не критично для работы комментариев

---

## ❌ Ошибка 4: Manifest warnings

**Причина:** Некорректный manifest.json.

**Решение:** Не критично для работы приложения.

## ✅ Проверка работы

После исправлений:

1. **Обновите страницу** (Ctrl+F5)
2. **Откройте консоль** (F12)
3. **Проверьте логи:**
   ```javascript
   // Должно быть:
   Comment system initialized ✅
   ```

4. **Попробуйте войти** через форму авторизации
5. **Откройте задачу** → проверьте блок комментариев

## 🐛 Если проблемы остались

### Полная очистка кэша:

```javascript
// В консоли браузера:
localStorage.clear();
sessionStorage.clear();
location.reload(true);
```

### Проверка файлов:

1. Убедитесь что comment_system.js подключен:
```html
<!-- В конце index.html -->
<script src="comment_system.js"></script>
```

2. Проверьте что функции глобальные:
```javascript
// В консоли:
console.log(typeof window.login);      // должно быть "function"
console.log(typeof sendTaskComment);    // должно быть "function"
console.log(typeof formatComment);      // должно быть "function"
```
