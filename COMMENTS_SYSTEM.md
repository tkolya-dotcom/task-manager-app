# Система комментариев для задач и монтажей

## Что сделано

Добавлена возможность оставлять комментарии к задачам и монтажам в формате чата.

### Функционал:
- ✅ Просмотр всех комментариев к задаче/монтажу
- ✅ Добавление нового комментария
- ✅ Удаление своих комментариев
- ✅ Автоматическая прокрутка к последнему комментарию
- ✅ Поддержка Enter для отправки
- ✅ Отображение автора и времени комментария

---

## Инструкция по установке

### 1. Выполните SQL миграцию

```sql
-- Скопируйте и выполните файл sql/add_comments.sql в Supabase SQL Editor
```

**Что создаст миграция:**
- Таблица `comments` для хранения комментариев
- Индексы для быстрого поиска
- RLS политики безопасности
- Триггер для обновления `updated_at`

После выполнения SQL миграции добавьте JavaScript функции и HTML элементы (будет добавлено автоматически в index.html).

## Как использовать

### В деталях задачи:
1. Откройте задачу → вкладка "Подробно"
2. Прокрутите вниз до блока "💬 Комментарии"
3. Введите текст в поле ввода
4. Нажмите "Отправить" или Enter

### В деталях монтажа:
Аналогично задачам - блок появится после установки UI.

## Структура таблицы comments

```sql
CREATE TABLE comments (
    id UUID PRIMARY KEY,
    resource_id UUID NOT NULL,        -- ID задачи или монтажа
    resource_type TEXT NOT NULL,       -- 'task' или 'installation'
    user_id UUID REFERENCES users(id), -- Автор комментария
    message TEXT NOT NULL,             -- Текст комментария
    created_at TIMESTAMPTZ,
    updated_at TIMESTAMPTZ
);
```

### Типы ресурсов:
- `'task'` - комментарий к задаче
- `'installation'` - комментарий к монтажу

## Безопасность

### RLS Политики:
- ✅ **Чтение**: Все авторизованные пользователи
- ✅ **Создание**: Все авторизованные пользователи
- ✅ **Редактирование**: Только автор комментария
- ✅ **Удаление**: Только автор комментария

## Решение проблем

### Комментарии не загружаются

**Проверьте выполнение миграции:**
```sql
SELECT * FROM comments LIMIT 5;
```

**Проверьте логи в консоли:**
```javascript
console.log('Comments loaded:', comments);
```

### Не удаляется комментарий

**Проверьте RLS политики:**
```sql
SELECT * FROM pg_policies WHERE tablename = 'comments';
```

### Комментарии не отправляются

**Проверьте права доступа:**
```sql
-- Должно вернуть true
SELECT auth.role() = 'authenticated';
```
